# 🎬 Movie Details Dashboard | Power BI Project

## 📌 Overview
The Movie Details Dashboard is an interactive Power BI project built to analyze movie industry data and generate meaningful business insights through data visualization techniques. This dashboard focuses on revenue trends, movie ratings, voting patterns, genre performance, and yearly movie distribution. The project was designed to improve analytical thinking, dashboard design, KPI reporting, and storytelling with data using Power BI.

---

## 🚀 Project Objectives
- Analyze movie performance using revenue and ratings
- Identify top-performing movie genres
- Track monthly revenue trends
- Compare yearly movie releases
- Build an interactive dashboard with filters and KPIs
- Practice real-world business intelligence reporting

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

---

## 📊 Dashboard Features
### ✔ KPIs
- Total Revenue
- Total Votes

### ✔ Visualizations
- Total Rating by Movie Title
- Total Revenue by Month
- Total Revenue by Genre
- Count of Movie ID by Genre
- Count of Movie ID by Year

### ✔ Interactive Filters
- Timeline Filter
- Language Filter

---

## 📈 Key Insights
- Action movies generated the highest total revenue.
- Revenue peaked during September compared to other months.
- Drama and Comedy genres contributed a large share of movie counts.
- Movie releases increased significantly after 2015.
- Certain movie titles achieved noticeably higher audience ratings than others.
