# Movie-Details-Dashboard-Power-BI-Data-Analytics-Project
Interactive Power BI dashboard analyzing movie revenue, ratings, votes, genres, release trends, and language distribution using data visualization and business intelligence techniques.

## 🎬 Movie Details Dashboard – Power BI Project
### 📌 Project Overview

This Power BI dashboard provides a complete analysis of movie performance using different business intelligence and data visualization techniques.
The dashboard helps identify revenue trends, top-rated movies, genre performance, yearly movie growth, and audience engagement through votes and ratings.

## 🚀 Objectives
Analyze movie industry trends using Power BI
Identify top-performing genres and movies
Track monthly revenue growth
Compare movie counts across years
Build an interactive and visually clean dashboard
## 📊 Dashboard Features
Total Revenue KPI
Total Votes KPI
Revenue Trend by Month
Top Rated Movies
Revenue Analysis by Genre
Movie Count by Genre
Movie Count by Year
Interactive Filters:
Timeline Filter
Language Filter
## 🛠 Tools & Technologies Used
Power BI
Power Query
Data Cleaning
Data Visualization
DAX Measures
Interactive Dashboard Design
## 📈 Key Insights
Action genre generated the highest total revenue
Revenue peaked during September
Drama and Comedy contributed major movie counts
Significant increase in movie releases after 2015
